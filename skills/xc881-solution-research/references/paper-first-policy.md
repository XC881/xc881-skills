# Paper-First Policy

Use for principle-level questions:

- algorithms
- distributed systems
- databases
- ML/AI methods
- cryptography
- consensus
- compiler/runtime behavior
- security models
- performance theory
- evaluation methodology

Process:

1. Search recent surveys when field is broad.
2. Search original papers for core concepts.
3. Check benchmark papers or reproducible repos for practical status.
4. Extract only decision-relevant claims.

Output:

```text
Paper evidence:
- Claim:
- Source:
- Why it matters:
- Limitation:
```
