# Research Source Taxonomy

Repository-level source map for maintainers.

## Academic / principle sources

arXiv, ACM, IEEE, USENIX, NeurIPS, ICML, ICLR, ACL, EMNLP, VLDB, SIGMOD, SOSP, OSDI, NSDI, Semantic Scholar, Google Scholar.

## Standards/specs

RFC/IETF, W3C/WHATWG, TC39, ISO/IEC summaries, CNCF specs, OpenAPI, AsyncAPI, GraphQL specs.

## Implementation sources

Official docs, release notes, changelogs, migration guides, API references, official examples, maintainer GitHub issues/PRs/discussions, benchmark repos.

## Security/ops sources

CVE/NVD, GHSA, OSV, CWE, OWASP, vendor advisories, incident postmortems.

## Community support

Stack Overflow, framework forums, public maintainer discussions, engineering blogs as supporting evidence only.
